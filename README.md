## Project Overview
This project models real-world e-commerce delivery challenges by integrating multiple algorithmic paradigms to optimize time, cost, and vehicle capacity. 

## Strategies Implemented
1. **Recurrence & Graph Modeling:** Distance matrices and recursive cost calculation for travel paths.
2. **Greedy & Dynamic Programming:** Solved parcel selection utilizing Fractional and 0/1 Knapsack logic to maximize delivery value under weight constraints.
3. **Graph Algorithms:** Built shortest emergency paths via Dijkstra and evaluated structural infrastructure costs using Kruskal's MST.
4. **Travelling Salesman Problem (TSP):** Computed the absolute optimal delivery route sequence and analyzed its NP-hard scalability limitations.
